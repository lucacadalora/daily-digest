Meta Tags on This Page:

The following meta tags have been added to enhance image previews:

  • og:image with version parameter
  • og:image:url
  • og:image:secure_url
  • og:image:type
  • og:image:width and og:image:height
  • twitter:card set to summary_large_image
  • twitter:image and twitter:image:src
  • twitter:url

These meta tags should help social media platforms display the image correctly in link previews. The version parameter in image URLs helps break cache and ensures the latest version is fetched.
