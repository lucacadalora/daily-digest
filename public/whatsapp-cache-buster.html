WhatsApp Cache Buster

Breaking WhatsApp's aggressive preview cache

Implementation Details

This page is designed to force WhatsApp to refresh its preview cache by using a unique URL parameter in the image.social URL:

<meta property="og:image" content="https://image.social/get?url=dailydigest.id/latest/china-steel-reform?whatsapp_cache_buster=18032025" />
Note: WhatsApp aggressively caches link previews, sometimes for days or weeks. This implementation adds a special parameter to force WhatsApp to see this as a unique image URL, bypassing its preview cache.

When you share this page on WhatsApp, it should generate a fresh preview using image.social. The special parameter is ignored by image.social but helps WhatsApp recognize this as a new image.
