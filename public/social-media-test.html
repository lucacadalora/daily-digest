Social Media Sharing Test Page

This page is designed to test our implementation of dynamic social media previews using image.social.

Test Configuration

We're using image.social to generate dynamic screenshots of our pages for social media sharing.

The URL format we're using is:

https://image.social/get?url=dailydigest.id/[path]&[platform-specific-parameters]&t=[timestamp]

Platform-Specific Parameters:

  • Twitter: &twitter=true
  • Facebook: &fb=true
  • Telegram: &telegram=true with more aggressive cache busting

Implementation Details

Our social media sharing strategy includes:

  1. Dynamic OpenGraph images using image.social
  2. Fallback to static images when required
  3. Special handling for newsletter links (no previews)
  4. Platform-specific optimizations (Twitter, Facebook, Telegram)
  5. Cache busting to ensure fresh previews

The core components include:

  • DynamicOGImage.tsx - React component for dynamic previews
  • GlobalOGImage.tsx - Global component applying to all pages
  • MetaTags.tsx - General meta tag management
  • ArticleSEO.tsx - Article-specific SEO
  • Server-side routes for handling social media crawler requests
