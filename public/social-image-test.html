Social Media Image Test

This page demonstrates our implementation of automatic social media previews using image.social.

How to test: Share this page on social media platforms like X (Twitter), WhatsApp, Facebook, or LinkedIn to see the dynamic preview image generated from this page.

Implementation Details

We're using image.social's URL-based approach to generate preview images. The key code is:

<meta property="og:image" content="https://image.social/get?url=dailydigest.id/social-image-test&viewport=1920x1080">

This approach works across all social platforms that support OpenGraph and Twitter Card meta tags.

Benefits of this approach

  • No manual image creation needed
  • Each page gets its own unique preview
  • Preview shows what's actually on the page
  • Automatic updates when page content changes
  • Works consistently across all platforms
