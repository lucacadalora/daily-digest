Daily Digest - Social Image Test

This page demonstrates the integration with image.social for automatic OpenGraph image generation.

1. Default Social Image

The default social image for dailydigest.id:

Meta tag: <meta property="og:image" content="https://image.social/get?url=dailydigest.id">

2. Article-specific Social Image

The social image for a specific article path:

Meta tag: <meta property="og:image" content="https://image.social/get?url=dailydigest.id/latest/china-steel-reform">

3. Test Your Own Path

Enter a path to test (without the domain):

Meta tag: <meta property="og:image" content="https://image.social/get?url=dailydigest.id">

How This Works

Image.social is a service that takes screenshots of your website to create beautiful, contextually relevant social media preview images.

  1. When your URL is shared on social media, the platform fetches the OpenGraph meta tags
  2. The og:image tag points to image.social with your URL as a parameter
  3. Image.social visits your page, takes a high-quality screenshot, optimizes it, and returns it
  4. The social platform displays this screenshot as the preview image

Benefits:

  • Automatic, high-quality social images for every page on your site
  • No manual image creation required
  • Preview shows actual page content for higher engagement
  • Retina-scaled for crisp display on all devices
  • Fast loading thanks to optimization
