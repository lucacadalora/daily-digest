Testing Twitter Card integration with image.social dynamic preview images

Implementation Details

This page is designed to test Twitter Card integration with image.social using the correct URL format:

https://image.social/get?url=dailydigest.id/latest/china-steel-reform&t=20250318

When this page is shared on Twitter/X, it should display a large preview image generated dynamically by image.social from the actual content of our website.

Key implementation notes:

  • Using twitter:card with value summary_large_image
  • The twitter:image meta tag uses the proper image.social URL format
  • A timestamp parameter is added to ensure fresh content from image.social
  • No Open Graph tags to ensure we're testing Twitter-specific implementation
